# Movie DB React

# Usage

First, clone this repository:

```bash
$ git clone https://github.com/zorfling/movie-db-react.git
```

Install dependencies:

```bash
$ yarn
```

Run for dev:

```bash
$ yarn start
```

Build:

```bash
$ yarn build
```

# Hosting

Hosted on Netlify at https://movie-db-react.netlify.com
